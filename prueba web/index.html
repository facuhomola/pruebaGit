Manejador de Eventos
onMouseOver (Pase el cursor del mouse aqui)
onMouseOut (Para ver el efecto ponga el cursor en este cuadro y luego alejelo)
onMouseMove (para ver efecto mueva el cursor sobre el cuadro)
onClic (para ver efecto haga clic en uno de los botones)
onwheel (Para ver este evento, mueva la rueda del raton sobre este cuadro)
onkeypress (Este evento se produce al presionar una tecla)
onkeydown (presione una tecla sobre la caja de texto para ver el efecto)
onkeyup (presione un caracter en miniscula para ver el efecto)
onmouseup y onmousedown

Haga clic para ver efecto
